# CLPT N5 (Fundamentos de Interfaz)

## Descripción del Nivel

El nivel N5 se enfoca exclusivamente en la estructura y el diseño estático de la información.

Al no usar JavaScript, el reto está en crear interfaces visualmente impactantes, perfectamente organizadas y accesibles utilizando solo la semántica de HTML5 y las capacidades de diseño de CSS3 (Flexbox, Grid, etc.).

## ¿Qué se va a Aprender en el Nivel?

* Uso de etiquetas semánticas de HTML5 para estructurar datos técnicos de manera jerárquica.

* Maquetación avanzada con CSS (diseño responsivo para pantallas de campo y monitores de centro de mando).

* Variables CSS para manejo de temas (ej. Modo Nocturno/Emergencia).

## ¿Con que Obejtivos?

* Construir dos proyectos en GitHub independientes con código limpio y validado.

* Implementar un diseño responsivo que se adapte a tablets de operarios en terreno y monitores de escritorio.

## ¿Qué Tecnologías se van a Usar?

* **HTML5** (Estructura pura).

* **CSS3** (Estilos, Flexbox/Grid, Animaciones básicas de transición, Media Queries).

  * Organizar grandes volúmenes de datos técnicos, tablas de especificaciones de hardware de robots y formularios de reporte de fallas estructuralmente impecables.
